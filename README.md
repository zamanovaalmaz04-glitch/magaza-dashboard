\# 🛒 Mağaza Satış Paneli



Pərakəndə mağazalar üçün real vaxt satış analitika dashboard-u.  

Excel faylını yüklə — dashboard saniyələr içində hazır olsun.



\## 🔴 Canlı Demo

👉 \[Dashboard-u aç](https://magaza-dashboard-ltjjl4hj9g7siy6kgwzh7n.streamlit.app/)



\---



\## 🎯 Hansı problemi həll edir?



Kiçik və orta mağazalar satış məlumatlarını Excel-də saxlayır,  

lakin faydalı qərar çıxarmaq üçün saatlarla vaxt xərcləyirlər.  



Bu dashboard ilə:

\- Aylıq gəlir trendini \*\*30 saniyədə\*\* görürsən

\- Ən çox satan məhsulları \*\*avtomatik\*\* sıralayırsan

\- Zəif kateqoriyaları \*\*dərhal\*\* aşkarlayırsan



\---



\## ✨ Xüsusiyyətlər



| Xüsusiyyət | Təsvir |

|---|---|

| 📂 Universal fayl dəstəyi | Excel (.xlsx, .xls) və CSV |

| 🧹 Avtomatik data təmizləmə | Boş hüceyrə, yanlış tarix, dublikat — avtomatik həll olunur |

| 📊 KPI kartları | Ümumi gəlir, əməliyyat sayı, orta çek |

| 📈 Trend analizi | Günlük və aylıq gəlir qrafikləri |

| 🏆 TOP 10 məhsul | Ən çox gəlir gətirən məhsullar |

| 🥧 Kateqoriya analizi | Hər kateqoriyanın ümumi gəlirdəki payı |

| ⚡ Real vaxt yeniləmə | Fayl yüklənən kimi dashboard dərhal yenilənir |

| ⚠️ Xəta idarəetməsi | Hər cür fayl formatı və data problemi avtomatik həll olunur |



\---



\## 🚀 İstifadə



\*\*1.\*\* Dashboard-da \*\*"Nümunə Excel yüklə"\*\* düyməsinə bas — formatı gör  

\*\*2.\*\* Öz Excel və ya CSV faylını yüklə  

\*\*3.\*\* Tarix, məbləğ, məhsul sütunlarını seç  

\*\*4.\*\* Dashboard dərhal yenilənir
